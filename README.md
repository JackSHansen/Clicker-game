# 🎮 Opgavebeskrivelse: Clicker Game Webapplikation
## 📌 Projektets titel

### Clicker Game – Interaktiv webapplikation udviklet i Next.js med TypeScript og SCSS

## 🧠 Problemstilling

Mange browserbaserede spil er enten for simple eller afhænger af eksterne frameworks og motorer. Dette projekt undersøger, hvordan man kan udvikle et fuldt funktionelt og engagerende clicker game ved hjælp af moderne webteknologier som Next.js, TypeScript og SCSS.

## 🎯 Formål

### Formålet med projektet er at:

Udvikle et interaktivt spil med moderne webframeworks
Lære at arbejde med Next.js (React-baseret framework)
Forstå brugen af TypeScript til type-sikkerhed i frontend udvikling
Implementere spil-logik, state management og brugerinteraktion
Arbejde med SCSS for struktureret og modulært design
Skabe en responsiv og brugervenlig webapplikation
## 🛠️ Teknologier

### Projektet udvikles med:

Next.js – framework til React-baseret webudvikling
TypeScript (TSX) – type-sikker JavaScript til bedre kodekvalitet
SCSS – avanceret styling med variabler og nesting
LocalStorage (eller alternativ state persistence) – til at gemme progression
## 🎮 Spilidé

Spillet er et klassisk clicker game, hvor spilleren optjener point ved at klikke på en knap.

Pointene kan bruges til at købe upgrades, som øger point-indsamlingen og skaber progression i spillet.

### Kernefunktioner:

Klik-knap der genererer point
Score- og progression system
Upgrades (fx +klik power, auto-click funktion)
Stigende priser for upgrades
Simple animationer for feedback
Gemning af progression via localStorage
## 🧩 Arkitektur og struktur (Next.js)

### Projektet struktureres som en moderne webapp:

pages/ eller app/ – sider og routing
components/ – UI komponenter (knapper, upgrades, HUD)
styles/ – SCSS filer (modulært design)
utils/ – spil-logik (point system, upgrades)
types/ – TypeScript interfaces
## 🚀 Udvidelsesmuligheder

### Hvis der er tid, kan projektet udvides med:

Animationer (fx Framer Motion)
Lyd-effekter ved klik og upgrades
Achievement system
Prestige/reset system med bonus
Dark/light theme
Leaderboard (lokal eller online database)
API-integration (fx daglige challenges)
## 📊 Mål

### Målet er at udvikle en fuldt funktionel webbaseret spilapplikation, der demonstrerer:

Forståelse af moderne frontend-arkitektur
Evne til at arbejde struktureret med komponenter
Grundlæggende spiludvikling i webmiljø
God UI/UX og brugeroplevelse
🧾 Konklusion (forventet)

Projektet skal resultere i et stabilt og interaktivt clicker game, udviklet med moderne webteknologier, hvor både funktionalitet, design og kodekvalitet er i fokus.
